Template Dataset

Unduh template atau contoh dataset di bawah ini, lalu isi data sesuai nama kolom yang sama agar file bisa langsung diproses oleh model.

Template Siap Isi Contoh Dataset

Format Dataset Manual yang Didukung

Wajib Ikuti Header
Kolom Inti Wajib Reporter, Partner, HS4, Year, Reporter_GDP, Partner_GDP, Reporter_Growth, Partner_Growth
  • Reporter: kode negara 3 huruf, contoh `IDN`.
  • Partner: kode negara 3 huruf, contoh `JPN`.
  • HS4: boleh `0101` atau `101`, sistem akan normalisasi ke format model.
  • Year: tahun numerik, contoh `2024`.
  • Reporter_GDP / Partner_GDP: isi nilai GDP numerik untuk reporter dan partner di tahun yang sama.
  • Reporter_Growth / Partner_Growth: isi nilai growth numerik untuk reporter dan partner di tahun yang sama.
  • Status / ExportValue: opsional. Kalau `Status` ada, baris non-Export akan diabaikan.
